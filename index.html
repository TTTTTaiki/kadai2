このページではHTMLの要素の中から３つを選んで説明を行います。

  • a要素
  • strong要素
  • img要素

  1. そもそも「要素」とは何か

    HTMLにおける「要素」とは、開始タグと終了タグで挟まれたものの全体を指します。
    例えば、「<p>文章A</p>」というようにコーディングした場合、 開始タグが「<p>」、終了タグが「</p>」、要素が「<p>文章A</p>」という事になります。

  2. strong要素

    strong要素を使うと、文字を太くすることができます。
    文字を強調したい場合に有効です。
    <strong></strong>タグで文字を挟むことで強調することができます。

    マークアップ例 :
    • 今日の朝ごはんは<strong>焼肉</strong>でした。
    • I have a <strong>PEN</strong>

    実行結果 :
    • 今日の朝ごはんは焼肉でした。
    • I have a PEN!

  3. img要素

    img要素を使うと、Webページに画像を追加することができます。
    この要素には終了タグが無く、文字列を囲まずに単独で使うのが特徴的です。
    例 : <img src="画像へのパス" alt="写真の説明" >
     画像を表示するには、「どの画像を表示するのか」という場所を指定するsrc属性、「どんな画像なのか」を指定するalt属性の二つを指定する必要があります。
    src属性にはその画像へのパス、alt属性にはその画像の簡単な説明を記述します。

    マークアップ例 :
    • <img src="./images/apple.JPG" alt="Apple" >
    • <img src="./images/pineapple.JPG" alt="Pineapple" >

  4. a要素

    a要素を使うと、他のWebページへのURLを貼り付けることができます。
    例 : <a href="表示したいWebページのURL">URLにしたい文字など</a>
    href属性には、表示したいWebページのURLを記述し、開始タグと終了タグの間に、URLにしたい文字などを入れます。 先に説明したimg要素を入れることで、画像にURLを埋め込むこともできます。

    マークアップ例 :
    • <a href="https://www.nintendo.co.jp/">任天堂ホームページ</a>
    • <a href="https://portal.educ.chs.nihon-u.ac.jp/"><img src="./images/CHS.jpg" alt="文理学部ポータルサイト"></a>

    実行結果 :
    • 任天堂ホームページ
